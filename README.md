# Vita-Guide

## Running the Client and Server

### Client
1. Navigate to the client directory:
    ```
    cd client
    ```
2. Install dependencies:
    ```
    npm install
    ```
3. Run the client:
    ```
    npm start
    ```

### Server
1. Navigate to the server directory:
    ```
    cd server
    ```
2. Install dependencies:
    ```
    npm install
    ```
3. Run the server:
    ```
    npm start
    ```
